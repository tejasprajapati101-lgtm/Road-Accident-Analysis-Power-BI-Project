# Road-Accident-Analysis-Power-BI-Project


This project focuses on analyzing road accident data to identify key patterns, trends, and insights that can help improve road safety.

 Tools & Technologies:
- Power BI (Dashboard & Visualization)
- Excel (Data Cleaning)
- SQL (Data Extraction)
- Python (EDA - Pandas, NumPy, Matplotlib)

 Key Analysis :
- Total number of accidents and casualties
- Monthly and yearly trends analysis
- Urban vs Rural accident comparison
- Severity analysis (Fatal, Serious, Slight)
- Vehicle type analysis

Key Insights:
- Identified high-risk areas and peak accident periods
- Found major causes contributing to accidents
- Suggested data-driven improvements for safety
